## Suggested Decision Rule

Use this rule when deciding what to do with each archive item:

- Include it if it adds practical guidance not already represented in a clearer form.
- Move it if it belongs in a different section but is still useful.
- Keep it archived if it is historical, redundant, or too experimental for the active system.

## Recommended Priority Order

1. AI operating layer framework and related assets
2. Legacy quality 101 decision artifacts
3. AI testing model materials for consolidation review
