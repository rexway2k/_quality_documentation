# L0 — Data Accuracy Testing

## What
Validates that data values correctly represent real-world facts.

---

## Why
Inaccurate data produces misleading outputs and incorrect reasoning.

---

## Where
- Training data
- Ground truth datasets
- External data sources

---

## When
- Data ingestion
- Periodic validation
- Before model training

---

## How
- Cross-source validation
- Manual verification
- Data reconciliation

---

## Failure Modes
- Incorrect values
- Stale or outdated truth

---

## Verification / Signals
- Accuracy rate vs trusted source
- Error rate
