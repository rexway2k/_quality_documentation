# Archive Review Mirror

## Purpose

This folder is a non-destructive mirror of the archived material so it can be reviewed side by side with the active quality management system without moving anything out of Archive_Save.

## What This Folder Contains

- a mirrored copy of the archive content under this folder
- a comparison index for likely matches with the active QMS

## Review Approach

Use this folder to:
- review archive content without changing the original archive
- compare archive content against the current QMS structure
- decide later whether any content should be incorporated, referenced, or left archived
