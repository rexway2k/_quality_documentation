# L4 — Data Flow & Mapping Testing

## What

Validates correct movement and transformation of data across systems.

---

## Why

Incorrect mapping breaks system behavior silently.

---

## Where

- Data pipelines
- API transformations
- Service layers

---

## When

- Integration setup
- Schema changes

---

## How

- Data mapping validation
- Schema comparison
- Transformation checks

---

## Failure Modes

- Field mismatch
- Missing data
- Incorrect transformation

---

## Verification / Signals

- Mapping accuracy rate
- Data validation errors
