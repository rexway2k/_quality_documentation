# L4 — Input / Output Handling Testing

## What

Validates system handling of inputs and outputs across boundaries.

---

## Why

Improper handling can break execution or cause errors.

---

## Where

- UI inputs
- API inputs/outputs
- System data transfers

---

## When

- Integration testing
- Data flow validation

---

## How

- Boundary testing
- Format validation
- Input sanitization checks

---

## Failure Modes

- Invalid input accepted
- Output misformatted
- Data loss

---

## Verification / Signals

- Input validation pass rate
- Output correctness rate
