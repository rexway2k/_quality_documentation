# L4 — Error Messaging & Handling Testing

## What

Validates how errors are surfaced to users and handled by the system.

---

## Why

Poor error handling reduces trust and usability.

---

## Where

- UI errors
- API errors
- System responses

---

## When

- Failure scenario testing
- UX validation

---

## How

- Negative testing
- Failure injection
- UX review

---

## Failure Modes

- Hidden errors
- Misleading messages
- No recovery guidance

---

## Verification / Signals

- Error visibility rate
- User recovery success rate
