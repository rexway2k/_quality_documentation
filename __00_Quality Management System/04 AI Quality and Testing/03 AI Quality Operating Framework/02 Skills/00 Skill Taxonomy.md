# Skill Taxonomy

## Purpose

This document defines the core skill areas that make up the AI Quality Operating Framework.

## Skill Layers

### Layer 0 — Principle Understanding
- interpret principles
- operationalize principles
- detect principle violations

### Layer 1 — Contract Definition
- define contract
- identify consumers
- define constraints
- define structure

### Layer 2 — Risk and Verification
- identify failure modes
- bound risk
- define verification
- determine execution readiness

### Layer 3 — Execution
- execute work
- maintain contract alignment
- surface assumptions
- handle ambiguity

### Layer 4 — Review and Feedback
- diagnose failure
- trace failure to contract
- validate output quality
- capture learnings
